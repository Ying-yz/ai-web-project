Tlias 智能学习辅助系统 (AI-Web-Project)
📝 项目简介
本项目是一个基于 Spring Boot 3 + MyBatis + Vue 3 架构开发的现代化全栈后台管理系统。项目采用前后端分离模式，集成了 JWT 安全认证、AOP 自动化日志监控、多表关联查询及复杂分页处理等核心业务场景。通过本项目，实现了对企业内部员工、部门、课程及操作日志的闭环管理。

🛠 技术栈
后端 (Backend): 基于 Java 17，使用 Spring Boot 3 作为核心框架。持久层采用 MyBatis，并集成 PageHelper 插件实现物理分页。使用 AOP 实现对系统操作的无侵入式监控记录。

前端 (Frontend): 基于 Vue 3 (Composition API) 结合 Element Plus UI 组件库，使用 Axios 进行前后端数据交互，实现响应式布局和流畅的用户体验。

数据库 (Database): 使用 MySQL 8.0 存储业务数据，设计了包括员工、部门、课程、操作日志在内的多张关联表。

🌟 核心功能特性
1. 自动化操作日志系统 (AOP & Logging)
全自动采集: 利用 Spring AOP 切面技术，自动拦截所有 Controller 层的增删改操作。

多表关联显示: 记录中自动关联 emp 表与 operate_log 表，实时解析并显示操作人的真实姓名，而非仅存储 ID。

详细数据追踪: 完整记录操作类名、方法名、请求参数、执行耗时以及方法返回值，便于系统审计与问题溯源。

2. 综合管理模块
部门管理: 支持部门信息的维护，支持按更新时间倒序排列显示，确保最新动态优先可见。

员工管理: 维护员工基本信息，并作为系统中所有操作行为的责任主体。

课程管理: 提供课程的新增、编辑、删除及状态管理（如：已结课等状态显示）。

3. 高效分页与查询优化
通用分页封装: 前后端配合实现高效的分页查询，通过 PageResult 对象统一封装总记录数和当前页数据。

多维度搜索: 支持根据名称、时间段等条件进行组合筛选，提升数据检索效率。

4. 系统安全性与鲁棒性
全域异常拦截: 配置全局异常处理器，捕获并统一处理系统运行时的各类异常，避免暴露底层堆栈信息。

前端状态管理: 具备完善的前端缓存处理逻辑，确保页面数据展示的准确性与实时性。
